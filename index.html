Codewars Fizz/Buzz 
                      
/*
    Write a function that takes an integer
    and returns an array [A, B, C].
    A is the number of multiples of 3 (but not 5) 
    below the given integer.
    B is the number of multiples of 5 (but not 3)
    below the given integer.
    C is the number of multiples of 3 and 5
    below the given integer.
    For example, solution(20) should return 
    [5, 2, 1]. 
*/    
    function solution(number) {
        const arr = [0, 0, 0];
        for (let i = 1; i < number; i++) {
            if (i % 3 === 0 && i % 5 === 0) {
                arr[2]++;
            } else if (i % 5 === 0) {
                arr[1]++;
            } else if (i % 3 === 0) {
                arr[0]++;
            }
        }
        return arr;
    }
